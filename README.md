    - 프로젝트 기대 효과

        - 관리자 기능을 통해 안전하고 신뢰할 수 있는 후원 환경을 제공합니다.
        - 간단하고 무결한 후원 과정을 통하여 후원에 대한 관심도를 증폭시킬 수 있을 것입니다.
